History Of internet

At the first stage of internet it was only used by Universities and Reasearch organisation in order to share information with each others and make some research. The ARPANET network expanded to quickly and by 1973 had grown to include more than 30 institutions encompassing government and private offices and organisations.

CSS

CSS (Cascsading Style Sheet) is a style sheet language used for describing the presentation of a document written in markup language such as html.Before CSS, document authors who wanted to assign such typographic characteristics to, say, all h2 headings had to repeat HTML presentational markup for each occurrence of that heading type. This made documents more complex, larger, and more error-prone and difficult to maintain

HTML

HTML (Hyper Text Markup Language) is used to define structure of web pages. It is the markup language which is used by web browsers to interpret text, images, and other material into visual or audible web pages. In html there are more 150 tags which we use to define various types of text, tables or grids. It can be assisted by javascript or cascading style sheets.
